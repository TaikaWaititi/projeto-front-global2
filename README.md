
# CareerMatch (frontend)
**Descrição**
O pont e objetivo do nosso projeto é criar um site onde as pessoas respondem uma série de perguntas e o site propõe uma série de carreiras para o usuário buscar descobrir mais e se modernizar na área de TI com a inteligência artificial

**Status do Projeto**
O status do projeto consta como um protótipo porém quando unificado as outras matérias ele se torna completo se ligando a banco para guardar informações de login, Java a API via URL, Python com as questões desenvolvidas e chatbot com a IA que realizará o quiz

**Sumário**
Escolhemos cores nos tons roxos pois aponta modernidade e tecnologia por ser uma cor próxima ao roxo e nosso site busca ser simples e útil para as pessoas experientes e as novatas no mercado de tecnologia

**Tecnologias Utilizadas**
- React 18
- Vite
- TypeScript
- TailwindCSS

**Como Usar**
O site é extremamente simples de usar, você cria um login para poder realizar o quiz e clica para responder as questões na página inicial, então o usuário será direcionado para a área de chatbot
